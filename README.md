# Sage Professional

A Palenight-inspired dark theme with sage green personality.

## Syntax Palette

| Token | Color | Hex |
|-------|-------|-----|
| Keywords | Purple | `#C792EA` |
| Functions | Blue | `#82AAFF` |
| Strings | Mint Green | `#C3E88D` |
| Constants | Gold | `#FFCB6B` |
| Operators | Cyan | `#89DDFF` |
| Tags | Coral | `#F07178` |
| Properties | Sage | `#A8D4C0` |
| Comments | Gray (italic) | `#546E7A` |

**Editor background:** `#1a2421` | **UI accent:** Sage green

## Installation

### From Marketplace

Search for "Sage Professional" in VS Code Extensions (`Cmd+Shift+X`).

### Manual

1. Copy the `jv-sage-vscode-theme` folder to `~/.vscode/extensions/`
2. Restart VS Code
3. `Cmd+K Cmd+T` and select **Sage Professional**
